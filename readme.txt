== Description ==

Weather Today Plugin by Creo is a Widget in your sidebar that will instantly tell you today’s weather condition. It shows Temperature, Sunrise/Sunset in Local Time (**Pro**), Humidity (Percentage), Wind speed and direction and Forecast. 

= Features =

Here are the amazing features:

* Super Easy installation
* Fully Responsive Widget
* Shows Temperature, Sunrise/Sunset in Local Time (**Pro**), Humidity (Percentage), Wind speed and direction and Forecast
* Uses HTML5 Location Service, down to 6 decimal points of Latitude & Longitude
* Fetches Weather Data from OpenWeather API, an open source weather API with fast frequency update.
* WordPress Color Chooser for Background and Font, Choose every COLOR available for Customization!
* Beautiful icons showing clear ideas of forecasting (**Pro**)
* Translation Ready. (An empty .po file with placeholders is given.)
* Best practices and guidelines for WordPress plugins are strictly followed, to ensure a consistent experience for site admin.
* Supports all modern browsers

== Installation ==

= For automatic installation: =

The simplest way to install is to click on 'Plugins' then 'Add' and type 'Creo Currency Converter Lite for Woocommerce' in the search field.

= For manual installation : =

1. Login to your website and go to the Plugins section of your admin panel.
1. Click the Add New button.
1. Under Install Plugins, click the Upload link.
1. Select the plugin zip file (woocommerce-currency-converter-lite.zip) from your computer then click the Install Now button.
1. You should see a message stating that the plugin was installed successfully.
1. Click the Activate Plugin link.
